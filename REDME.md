# Introduction to Algorithms

## Getting Started

This is a code repository in C++ programming language about the examples in the Book with the same name by Thomas H. Cormen, *et al.*

The pseudocode are in the book if you want to make the algorithms in already other programming language as Python, Java, etc.

## Folder structure

You find every example in a folder as the same name of the chapter in the book and every example are in a .cpp file with the same name as the listing title in the book
